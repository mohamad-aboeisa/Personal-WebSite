# Personal Website

A personal website project built while practicing modern front-end development fundamentals using HTML and CSS. The project focuses on creating clean layouts, responsive sections, and improving overall page structure and styling techniques.

Through this project, I practiced working with Flexbox, CSS Grid, responsive design principles, and reusable styling patterns to create a website that adapts smoothly across different screen sizes and devices.

## Technologies Used
- HTML5
- CSS3
- Flexbox
- CSS Grid
- Responsive Design
